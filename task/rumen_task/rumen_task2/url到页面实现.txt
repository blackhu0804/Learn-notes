从url输入到页面展现

1、输入url：统一资源定位器，定位互联网上的资源。
https比http更安全。
2、域名解析，查找域名代替的ip地址
	域名解析的流程：
     		1、浏览器缓存 C 浏览器会缓存DNS记录一段时间
		2、系统缓存 - 从 Hosts 文件查找是否有该域名和对应 IP。
		3、路由器缓存 C 一般路由器也会缓存域名信息。
	       	4、ISP DNS 缓存 C 比如到电信的 DNS 上查找缓存。
		5、如果都没有找到，则向根域名服务器查找域名对应 IP，根域名服务器把请求转发到下一级，直到找到 IP

3、web服务器接收用户的request交给网站代码，或者接受请求反向代理到其他服务器。
4、mvc反馈
