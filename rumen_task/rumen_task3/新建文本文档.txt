git的三个重要概念 ：
已提交。
已修改。 git add -> 已暂存。



git push -f origin master //强制推送，会覆盖别人代码
git remote set-url origin 地址 //修改origin对应的地址


分支操作
git branch -a //查看本地分支
git branch name //创建新的分支
git checkout name //切换到name分支

切换到master
